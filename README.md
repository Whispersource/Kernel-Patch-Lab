## 🚀 项目简介

`Kernel-Patch-Lab` 是一个实验性的仓库，专注于收集和开发针对 Android 设备 Linux 内核的补丁和工具。本项目旨在深入研究内核机制，模拟对 APatch、KernelSU 等内核级 Root 解决方案的理解和贡献，并探索自定义内核功能。

### ✨ 主要特性

*   **内核补丁集合**: 包含针对不同 Android 版本和设备模型的内核补丁示例，例如 SELinux 策略调整、自定义系统调用等。
*   **内核模块开发**: 探索 Linux 内核模块 (LKM) 的开发与加载，用于实现系统级功能扩展。
*   **Root 方案研究**: 模拟对现有内核级 Root 方案（如 KernelSU）的原理分析和功能增强。
*   **C 语言实现**: 内核相关代码主要使用 C 语言编写。
*   **自动化脚本**: 提供用于应用和管理内核补丁的辅助脚本。

## ⚙️ 构建与使用

1.  **克隆仓库**:
    ```bash
    git clone https://github.com/Whispersource/Kernel-Patch-Lab.git
    cd Kernel-Patch-Lab
    ```
2.  **准备环境**: 需要具备 Linux 内核编译环境和 Android AOSP 源码。
3.  **应用补丁**: 参考 `patches/` 目录下的补丁文件，使用 `git apply` 或 `patch` 命令将其应用到目标内核源码。
4.  **编译内核**: 编译并刷写修改后的内核到您的 Android 设备。
